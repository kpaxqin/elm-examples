# elm-examples

Examples for learning [elm](http://elm-lang.org/)

Run `npm start` to start

## Articles to learn elm

Basic syntax : the [learn x in y minutes](https://learnxinyminutes.com/docs/elm/) looks even better than the [official syntax doc](http://elm-lang.org/docs/syntax#type-annotations)

For developers from JS : The [elm-for-js](https://github.com/elm-guides/elm-for-js) explains lot of things that really matters like Type and Scope for JS people
